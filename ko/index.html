자주 묻는 질문

PickUp Sign은 어떻게 사용하나요?

앱을 열고, 이름을 입력하고, '표시'를 탭하면 끝. 스마트폰이 이름을 최대한 크게 표시하는 전체 화면 사인으로 변신. 화면은 꺼지지 않고 밝기는 자동으로 최대가 됩니다.

오프라인에서도 작동하나요?

네. PickUp Sign은 완전한 오프라인으로 작동합니다. 인터넷 연결 불필요. 모든 처리는 기기 내에서 로컬로 이루어집니다.

화면이 꺼지지 않나요?

네. 이름을 표시하는 동안 화면은 계속 켜져 있고 밝기는 최대로 설정됩니다. 화면을 유지하기 위해 탭할 필요가 전혀 없어요.

배경 색상은 어떻게 바꾸나요?

화면 하단의 색상 선택 아이콘을 탭. 흰색과 검정은 무료. $0.99/year 색상 테마 업그레이드로 22가지 색상 전부 사용 가능.

어떤 언어를 지원하나요?

PickUp Sign은 12개 언어를 지원합니다. 영어, 스페인어, 프랑스어, 독일어, 이탈리아어, 포르투갈어, 일본어, 한국어, 중국어(간체·번체), 아랍어, 힌디어. 스마트 줄바꿈은 지원하는 모든 문자에서 작동합니다.

PickUp Sign은 무료인가요?

네. 핵심 기능(이름 입력, 전체 화면 표시, 화면 상시 켜짐, 밝기 최대)은 완전히 무료. 선택적 색상 테마 업그레이드($0.99/year)로 22가지 프리미엄 배경 색상을 사용할 수 있습니다.

구매를 복원하려면 어떻게 하나요?

앱을 열고 색상 선택 아이콘을 탭한 후 '구매 복원'을 탭. 원래 구매한 위치에 따라 Apple ID 또는 Google 계정을 통해 자동으로 복원됩니다.

iPad에서도 작동하나요?

네. PickUp Sign은 iPad를 완벽 지원하며 더 큰 화면을 최대한 활용해 이름을 더 크고 선명하게 표시합니다.

Android는요?

네. PickUp Sign은 Android에서 사용 가능합니다. Google Play에서 다운로드하세요.

PickUp Sign이 내 데이터를 수집하나요?

아니요. PickUp Sign은 개인 데이터를 전혀 수집하지 않습니다. 입력된 이름은 기기에만 저장되며 어떤 서버에도 전송되지 않습니다. 분석 없음, 추적 없음, 광고 없음. 개인정보 처리방침

광고가 있나요?

아니요. PickUp Sign에는 광고가 없고 앞으로도 절대 없을 거예요. 공항에서 사인 들고 있는데 배너 광고가 이름을 가리면 최악이잖아요. 화면에 보이는 건 사인뿐입니다.

다른 네임 사인 앱을 쓰면 안 되나요?

대부분의 네임 사인 앱들이 중요한 두 가지를 놓칩니다. 사인을 들고 있는 동안 화면이 어두워지고 잠기는 것, 그리고 밝기를 최대화하지 않는 것. PickUp Sign은 이름을 표시하는 순간부터 자동으로 화면을 계속 켜두고 밝기를 100%로 올립니다. 설정 건드릴 필요 없음, 픽업 도중 화면 꺼지는 당황스러운 일 없음. 광고도 제로. 그냥 이름만, 전체 화면으로.
